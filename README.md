# webserver
A simple webserver

### Requirements
1. A POSIX machine (Linux or FreeBSD)
2. gcc or clang (C++ 17)
3. cmake 2.23

### To run
1. execute `./runcmake.sh` to generate cmake files.
2. execute `./runbuild.sh`
3. run server by `./runserver.sh`
4. navigate to http://localhost:8000.

### Supports
1. HTML
2. CSS
3. Javascript
4. PNG/JPG
